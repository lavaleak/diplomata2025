## Development (How to Contribute)
### Dependencies
- **For Prisma ORM and scripts:**
    - node
    - npm
    - pnpm
- **For .NET dynamic library:**
    - dotnet (netcore)
- **For Application Cross-OS UI:**
    - Godot Engine (with .NET)
